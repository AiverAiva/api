# HOW TO USE?

### Base url: https://osu-stats-kappa.vercel.app/api/

## User

### osu! profile image

##### Overview
Returning an image of personal dat

###### URL
    /profile/{ username or id }.png
###### Parameters
- `mode` - mode (0 = osu!, 1 = Taiko, 2 = CtB, 3 = osu!mania). Optional
